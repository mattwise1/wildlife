
## 🚀 How to Run

1. Download or clone this project.
2. Extract the files into a folder.
3. Open any of the `.html` files in your browser (recommended: start with `index.html`).
4. No server required — works directly with local files.

---

## 🛠 Technologies Used

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript**
- **Font Awesome** icons
- Google Fonts (Playfair Display + Inter)

---

## 🎯 Main Features Explained

### Cart System
- Add tours from Home or Bookings page
- Cart persists even if you change pages
- Click cart icon → Modal opens showing items and total
- Remove items individually
- Simulated checkout with payment options

### Responsive Design
Fully optimized for **desktop, tablet, and mobile** devices.
